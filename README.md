# Plastic Scanner

An open-source plastic scanner project for material identification and analysis.

## Project Structure

- **CAD/** – Mechanical design files and enclosures
- **PCB/** – Electronics schematics and PCB layouts
- **Firmware/** – Embedded firmware and software
